# flutter, dart, huffman-coding, lzw, data-compression, algorithm-visualization, huffman-tree, mobile-app, compression-ratio
A Flutter mobile app for visualizing lossless data compression algorithms. 
Supports Huffman Coding (with custom-painted binary tree and step-by-step 
code table) and LZW compression (with dictionary build animation). Users 
can input custom text and see compression ratio stats in real time.
